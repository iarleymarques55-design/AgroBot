Atue como um Arquiteto de Software Sênior e Desenvolvedor Full Stack especialista em Inteligência Artificial e Agronegócio.

Quero desenvolver uma aplicação web profissional chamada AgroAI, um assistente agrícola inteligente que 
ajuda agricultores a tirar dúvidas sobre cultivo, solo, irrigação, fertilização, pragas, doenças e produtividade.

TECNOLOGIAS OBRIGATÓRIAS

Frontend:
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Interface moderna e responsiva

Backend:
- Node.js
- Express.js
- API REST

Banco de Dados:
- PostgreSQL
- Hospedado na Neon

Hospedagem:
- Frontend na Vercel
- Backend na Render
- Banco PostgreSQL na Neon
